Crystals Game

Can you add the crystals hidden values to equal the number given?

In this game you'll use simple addition to try to equal the computers random number.

When you click one of the crystals below the value will be shown in your score box.

Each of the crystals have a different value, using the different values try to add up to the number shown.

If you go over the value of the number shown you lose.

If you add the number correctly, you will win.

Win or lose the game will reset. You will be shown a new number to guess, and the crystals will have different values.
